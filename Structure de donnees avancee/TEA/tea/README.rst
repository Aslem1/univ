==========

Installation
------------

.. code:: bash

   $ sudo apt install cmake gcc

Usage
-----

Compilation
~~~~~~~~~~~

.. code:: bash

   $ cd debug
   $ cmake ../
   $ make

Run test/main
~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code:: bash

   $ ./build/src/treeRB/main


Install package
~~~~~~~~~~~~~~~

.. code:: bash

   $ make install
