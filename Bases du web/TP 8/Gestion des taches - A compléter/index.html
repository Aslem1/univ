Dans la page exercice, vous retrouverez l'exemple des tâches vu en cours.

Vous devez reproduire le fonctionnement décrit en cours.

Vous pouvez suivre ce plan :

  1. Rendre tous les boutons X réactifs. Vous devrez utiliser l'une des boucles vues en cours
  2. Faire en sorte qu'un clic sur un bouton X supprime ce bouton. Vous utiliserez la méthode remove et l'objet this
  3. Faire en sorte qu'un clic sur un bouton X supprime la tâche concernée. Vous devrez sélectionner un ascendant de l'objet this
  4. Traiter l'événement de validation du formulaire
  5. Supprimer le comportement par défaut de cet événement avec preventDefault()
  6. Traiter cet événement en récupérant les valeurs des champs qui et quoi
  7. Injecter ces valeur dans la tâche invisible
  8. Dupliquer la tâche invisible, rendre son clone visible et l'insérer dans la liste des tâches à effectuer
  9. Placer les écouteurs d'événement sur les deux boutons dela nouvelle tâche
  10. Gérer le bouton V : il doit permettre de transférer la tâche dans la liste des tâches accomplies
